# README #

STAR WARS APP WITH SWAPI API

### FEATURES ###

* LOGIN WITH REDUX-FORM
* TYPE-ALONG SEARCH WITH DEBOUCE AND THROTTLE

### How do I get set up? ###

* Clone the repo
* npm i
* npm run build [for prod]
* npm run build:dev [for dev]
* npm start
* npm test [for tests]
